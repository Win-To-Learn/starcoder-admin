## Building

Make sure you have angular-cli installed globally:

```
npm install -g @angular/cli
```

Then:

```
npm install
ng build
```

## Serving

```
node src/server/index.js
```

Runs on localhost port 4200

## Status

Right now it runs against a couple of test collections on the mongolab db.
